About Calc.nz

An independent collection of guided NCEA maths worked-answer walkthroughs.

← Home

Purpose and approach

Learning the reasoning behind each step

Calc.nz is designed to help students attempt an exam-style question, request a hint when needed, and then work through the decisions and mathematical reasoning behind a strong response. The walkthroughs are learning aids, not a substitute for teaching, an official assessment schedule, or personalised academic advice.

The supported standards are:

  • NCEA Level 2 Calculus — AS91262
  • NCEA Level 2 Algebra — AS91261
  • NCEA Level 3 Complex Numbers — AS91577
  • NCEA Level 3 Differentiation — AS91578
  • NCEA Level 3 Integration — AS91579

Project information

Who made Calc.nz

Calc.nz was created by Jack van Baalen as part of a Year 13 extended learning project. Mathematical explanations and walkthrough design are by Jack van Baalen. AI tools were used to assist with parts of the website implementation.

The site does not claim that its walkthroughs are NZQA-verified, teacher-reviewed, or a replacement for official material.

Source policy and disclaimer

Check official assessment material

Standard titles and links point to NZQA. Question labels and walkthrough topics are organised to match the examination material represented in this project. Students and teachers should use the linked NZQA pages for original questions, diagrams, schedules, current standard information, and authoritative assessment requirements.

Calc.nz is independent and is not affiliated with or endorsed by NZQA. Any error in a walkthrough belongs to this project, not NZQA.

Privacy

A static learning website

Calc.nz does not add analytics, advertising, user accounts, mailing lists, or tracking cookies. Walkthrough progress, bookmarks, retry marks, display preferences, and practice sets may be stored only in the browser on the student's own device.

Students can clear this local information from the homepage. If browser storage is blocked, core pages and walkthroughs still work, but saved practice features last only for the current visit.

Corrections

Report an error

If you find a mathematical, wording, accessibility, or technical problem, use the Calc.nz error-report form. Include the page URL, question number, and a concise description so it can be checked.
