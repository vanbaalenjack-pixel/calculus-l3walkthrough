Question

The graph below shows the function \(y=f(x)\).

\[ \text{(i) Find the value(s) of } x \text{ where } f(x) \text{ is not differentiable.} \] \[ \text{(ii) Find the value(s) of } x \text{ for which } f'(x)=0. \] \[ \text{(iii) What is the value of } \lim_{x\to -4} f(x)\text{?} \]

Try the question yourself first. If you get stuck, open the hints before using the full walkthrough.

Step 1

Where is the graph not differentiable?

Look for holes, sharp corners, or jumps.

Step 2

Where is the derivative zero?

The derivative is zero where the graph has a horizontal tangent or is flat.

Step 3

Find the limit at \(x=-4\)

Even though the point is missing, follow the graph from both sides.

Step 4

Final answers

\[ \text{(i) } f(x)\text{ is not differentiable at } x=-4,\,-1,\text{ and }1 \] \[ \text{(ii) } f'(x)=0 \text{ when } x=-3 \text{ and for all } x>1 \] \[ \text{(iii) } \lim_{x\to -4} f(x)=3 \]

At \(x=-4\) there is a hole, so the function is not defined there. At \(x=-1\) and \(x=1\) there are sharp corners, so the derivative does not exist. The graph is flat at the turning point \(x=-3\) and on the horizontal ray for \(x>1\).
