Review the method, not only the answer

This walkthrough focuses on differentiating, evaluating a rate, and using derivatives to justify the minimum turning point. Afterwards, try the same method again without hints and explain the reasoning that connects each step.

Common mistake to avoid

Finding a stationary value is only part of the argument; justify that it is the required minimum and respect the domain.
