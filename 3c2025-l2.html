Learning summary

Review the method, not only the answer

This walkthrough focuses on using a surface-area constraint, writing volume in one variable, maximizing it, and proving the maximum. Afterwards, try the same method again without hints and explain the reasoning that connects each step.

Common mistake to avoid

Finding a stationary value is only part of an optimisation argument; justify that it is the required maximum and respect the domain.
