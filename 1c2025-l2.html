Question

\[ f'(x)=2x^2+2x-24 \]

Find the equation of the original function \(f(x)\) if the \(y\)-value of the local minimum is \(-46\).

Learning summary

Review the method, not only the answer

This walkthrough focuses on finding critical points, identifying the minimum, and solving for the constant. Afterwards, try the same method again without hints and explain the reasoning that connects each step.

Common mistake to avoid

Finding a stationary value is only part of the argument; justify that it is the required minimum and respect the domain.
