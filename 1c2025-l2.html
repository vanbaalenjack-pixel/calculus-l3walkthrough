Question

\[ f'(x)=2x^2+2x-24 \] \[ \text{Find the equation of the original function } f(x), \text{ if the } y\text{-value of the local minimum is } -46. \]

Try the question yourself first. If you get stuck, open the hints before using the full walkthrough.

Use the derivative to find the critical points, use the second derivative to identify the minimum, then integrate and solve for the constant.

Step 1

Find the critical points

Set the derivative equal to zero and solve.

Step 2

Find the second derivative

Differentiate \(f'(x)=2x^2+2x-24\) one more time.

Step 3

Identify the local minimum

Which critical point gives a local minimum?

Step 4

Find the original function

Integrate the gradient function to get \(f(x)\).

Step 5

Use the minimum value to find \(c\)

The local minimum happens at \(x=3\), and its \(y\)-value is \(-46\). Which equation should we use?

Step 6

Final answer

Now solve for \(c\) and write the full function.

\[ f(x)=\frac{2}{3}x^3+x^2-24x+c \] \[ f(3)=\frac{2}{3}(27)+9-72+c=-46 \] \[ 18+9-72+c=-46 \] \[ -45+c=-46 \] \[ c=-1 \] \[ f(x)=\frac{2}{3}x^3+x^2-24x-1 \]
Great work. The original function is \(f(x)=\frac{2}{3}x^3+x^2-24x-1\).
