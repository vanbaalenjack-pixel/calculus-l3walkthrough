Question

The curve with equation \((y-5)^2=16(x-2)\) has a tangent of gradient \(1\) at point \(P\).

This tangent intersects the \(x\)- and \(y\)-axes at points \(R\) and \(S\) respectively.

\[ \text{Prove that the length } RS \text{ is } 7\sqrt{2}. \]

You must use calculus and show any derivatives that you need when solving this problem.

Both implicit differentiation and rearranging to the upper branch \(y=5+4\sqrt{x-2}\) are valid methods here.

Try the question yourself first. If you get stuck, open the hints before using the full walkthrough.

Step 1

Differentiate implicitly

Differentiate both sides of \((y-5)^2=16(x-2)\) with respect to \(x\).

Step 2

Use the gradient information

Rearrange for \(\frac{dy}{dx}\), then use the fact that the tangent gradient at \(P\) is \(1\).

Step 3

Find point \(P\)

Solve for \(y\), then substitute back into the curve equation to find \(x\).

Preview will appear here.

Step 4

Find the tangent equation

Use gradient \(1\) and point \(P=(6,13)\).

Preview will appear here.

Step 5

Find the intercepts

Rewrite the tangent line and find where it meets each axis.

\[ y=x+7 \] \[ S=(0,7) \] \[ R=(-7,0) \]
Great. Now we can find the distance between \(R\) and \(S\).

Step 6

Prove the distance

Use the distance formula or Pythagoras on the intercept triangle.

\[ RS=\sqrt{(0-(-7))^2+(7-0)^2} \] \[ RS=\sqrt{7^2+7^2} \]
