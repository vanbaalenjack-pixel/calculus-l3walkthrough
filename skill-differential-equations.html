Skill overview

What these questions practise

The matched Integration methods explicitly name separation or a differential equation, while matched Differentiation methods verify or prove a differential-equation relationship.

Relevant standards: NCEA Level 3 Differentiation — AS91578 · NCEA Level 3 Integration — AS91579

This collection contains 23 matching walkthroughs across 9 examination years.

The grouping is based on the method recorded for each walkthrough. Use the linked NZQA standard and paper material for authoritative wording, diagrams, and assessment information.

Before you start

Common mistakes in these methods

  • Separate the variables consistently before integrating, without dropping factors that belong with either differential.
  • Retain the integration constant and apply the supplied condition, then check the resulting branch and domain in context.
  • For verification, calculate the required derivatives and substitute them into both sides of the stated differential equation.
