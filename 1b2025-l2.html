Review the method, not only the answer

This walkthrough helps you practise finding the point of tangency, the gradient, and the tangent equation. Use the hints to plan the method, then repeat the question without hints and check each step.

Common mistake to avoid

Use the derivative for the gradient and the original curve for the point before forming the tangent equation.
