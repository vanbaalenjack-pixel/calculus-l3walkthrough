Question

Find the gradient of the tangent to the curve \(y=(3x^2-2)^3\) when \(x=2\).

You must use calculus and show any derivatives that you need when solving this problem.

Try the question yourself first. If you get stuck, open the hints before using the full walkthrough.

Step 1

Choose the rule

This is a function raised to a power, so which differentiation rule is the best starting point?

Step 2

Differentiate the outside first

If \(u=3x^2-2\), then \(y=u^3\). What is \(\frac{dy}{du}\)?

Step 3

Differentiate the inside function

Now find \(\frac{du}{dx}\) for \(u=3x^2-2\).

Step 4

Combine with the chain rule

Multiply the outside derivative by the inside derivative.

Step 5

Substitute \(x=2\)

Now evaluate the derivative to find the tangent gradient.

\[ \frac{dy}{dx}=18x(3x^2-2)^2 \] \[ \frac{dy}{dx}\bigg|_{x=2}=18(2)\bigl(3(2)^2-2\bigr)^2 \]
