Question

\[ \text{Differentiate } f(x)=(5x-3)\sin(4x). \]

You do not need to simplify your answer.

Try the question yourself first. If you get stuck, open the hints before using the full walkthrough.

Step 1

Choose the differentiation technique

This function is one factor multiplied by another, so which rule should we start with?

Step 2

Differentiate the first factor

If \(u=5x-3\), what is \(u'\)?

Preview will appear here.

Step 3

Differentiate the sine factor carefully

If \(v=\sin(4x)\), what is \(v'\)? Remember the chain rule inside the sine function.

Preview will appear here.

Step 4

Assemble the product rule

Use \(\frac{d}{dx}(uv)=u'v+uv'\).

Preview will appear here.

Step 5

Final derivative and graph

A complete derivative is:

\[ f'(x)=5\sin(4x)+4(5x-3)\cos(4x) \]

Common mistake to avoid
